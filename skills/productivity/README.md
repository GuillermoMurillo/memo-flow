# productivity

Promoted productivity skills. Listed in root `README.md` and `.claude-plugin/plugin.json`.

- [grill-me](grill-me/SKILL.md): interrogate the user's plan with questions until alignment is reached. Wrapper over `/grilling`.
- [grilling](grilling/SKILL.md): the interview engine — relentless one-question-at-a-time grilling of a plan or design.
- [handoff](handoff/SKILL.md): write a session-ending handoff note (to `mktemp` path) so a fresh session can resume.
- [pager](pager/SKILL.md): portable display mode for small screens, streams to glasses or Remote Control.
- [writing-great-skills](writing-great-skills/SKILL.md): reference for writing and editing skills — the vocabulary and principles that make a skill predictable.
